# 🌙 Ramadhan Portal 2026

---

## 📝 Deskripsi Singkat
Selain bertujuan untuk menjadi asisten digital pribadi bagi user, aplikasi ini juga dibuat untuk mengisi waktu luang developer selama bulan ramadhan. Aplikasi ini memiliki 5  fitur utama dengan navigasi yang memudahkan pengguna berpindah halaman.

## 🛠️ Tech Stack
Proyek ini dibangun menggunakan teknologi modern yang ringan namun kuat:
* **HTML5**: Struktur semantik untuk konten web.
* **Tailwind CSS**: Framework CSS berbasis utilitas untuk desain UI yang responsif dan estetik.
* **JavaScript (ES6+)**: Logika aplikasi, manipulasi DOM, dan pengelolaan data dinamis.

## ✨ Fitur Utama
1.  **Jadwal Imsakiyah**: Menampilkan waktu shalat dan imsakiyah secara akurat berdasarkan lokasi pengguna (via API).
2.  **Kalkulator Zakat**: Modul perhitungan Zakat Penghasilan dan Zakat Emas sesuai dengan nisab harga emas terkini.
3.  **Tasbih Digital**: Counter dzikir interaktif dengan fitur target kustom (11, 33, 99) dan notifikasi saat target tercapai.
4.  **Daily To-Do List**: Checklist progres harian yang mencakup Shalat Wajib, Tilawah Quran (dengan target halaman), Puasa, dan Dzikir.
5.  **Navigation Universal**: Sistem navigasi responsif yang memudahkan berpindah antar fitur di perangkat mobile maupun desktop.

## 🚀 Cara Pakai (Usage)

### Menjalankan Secara Lokal
1.  **Clone Repositori**:
    ```bash
    git clone [https://github.com/username-kamu/ramadhan-portal.git](https://github.com/username-kamu/ramadhan-portal.git)
    ```
2.  **Buka Folder Proyek**:
    Masuk ke direktori proyek melalui terminal atau file explorer.
3.  **Jalankan di Browser**:
    Klik dua kali pada file `index.html` atau klik kanan dan pilih **Open with Live Server** jika menggunakan VS Code.

## Cara Penggunaan
Cukup buka file `index.html` di browser kamu atau akses melalui [https://quishasya.github.io/Alhazen-RCC/].

## 🧪 Optimasi & Testing
Web ini telah diuji dan dioptimasi untuk memastikan performa yang stabil:
* **Browser Compatibility**: Berjalan dengan mulus di **Google Chrome**, **Safari**.
* **Responsive Design**: Layout telah diuji pada berbagai ukuran layar, mulai dari smartphone (450px) hingga monitor desktop (1400px).
* **Performance**: Meminimalkan penggunaan library eksternal yang berat untuk memastikan waktu pemuatan (loading) yang sangat cepat.
